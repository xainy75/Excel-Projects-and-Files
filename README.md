# Excel Projects and Files

A curated collection of Excel practice workbooks and mini‑project assets focused on practical, real‑world spreadsheet skills.

## Overview

This repository contains `.xlsx` workbooks you can use to learn, practice, and demonstrate common Excel techniques, including:

- Data cleaning, formatting, and presentation
- Conditional formatting rules and best practices
- PivotTables and summary reporting
- Charts and basic data visualization
- Simple dashboards and worksheet organization

## Included files (what to practice)

All practice workbooks are located in `excel-files-folder/`.

- **Conditional Formatting - Participants File.xlsx**
  - Focus: building and managing conditional formatting rules (highlighting, data bars, icon sets, rule ordering).

- **Pivot Table - Participants File.xlsx**
  - Focus: creating PivotTables, grouping, sorting/filtering, and producing quick summaries.

- **Charts - Participants File.xlsx**
  - Focus: chart creation and formatting (choosing the right chart type, labels, axes, and presentation).

- **Dashboard - Participants File.xlsx**
  - Focus: assembling a simple dashboard (layout, KPIs, visuals, slicers/filters where applicable).

## Getting started

1. Clone or download the repository.
2. Open any workbook from `excel-files-folder/` using:
   - Microsoft Excel (recommended)
   - LibreOffice Calc or Google Sheets (features may vary)
3. Save a copy before editing so you can retry the exercises.

## Suggested workflow

- Start with **Conditional Formatting**, then move to **Pivot Table** and **Charts**.
- Use **Dashboard** last to combine skills.
- As you practice, try to:
  - Keep raw data separate from reporting sheets
  - Use consistent number/date formats
  - Prefer Tables (Ctrl/Cmd + T) for structured data
