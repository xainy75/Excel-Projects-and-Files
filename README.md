# Excel Projects and Files

A curated collection of Excel practice files and mini-project assets focused on real-world spreadsheet skills.

## 📌 Overview

This repository contains Excel workbooks intended for learning, practicing, and demonstrating common spreadsheet techniques such as:

- Data formatting and presentation
- Conditional formatting
- Practical worksheet organization
- Beginner-to-intermediate Excel workflows

## 📂 Repository Structure

```text
Excel-Projects-and-Files/
├── README.md
└── excel-files-folder/
    └── 41-42 - Conditional Formatting - Participants File.xlsx
```

## 🚀 Getting Started

1. Clone or download this repository.
2. Open the `.xlsx` files from `excel-files-folder/` using:
   - Microsoft Excel (recommended), or
   - Compatible spreadsheet tools (e.g., LibreOffice Calc, Google Sheets).
3. Explore each workbook and practice the included concepts.

## ✅ Included Files

  A practice workbook centered on conditional formatting scenarios.
